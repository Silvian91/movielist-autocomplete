# Movie list Auto-complete
Movie list auto-complete challenge

## Installation and running

Basic installment of node modules

```
npm install
```
Running the App locally

```
npm start
```

## Use Cases

* Selecting a movie title that starts with specific string.
* Navigating through the list of available movies with ```Arrow``` keys.
* Open the movie IMDB link by pressing  ```Enter```.
* Some crazy stuff I haven't thought about.
